## About Becwork
Becwork is a simple php framework.

## Becwork features
 - Included composer
 - APP panel
 - Simple config system
 - Simple tests system
 - Crypt utils
 - Hash utils
 - Mysql utils
 - Cookie utils
 - Session utils

## Becwork required
 - Set web root to web dir public/
 - Disable apache autoindex
 - php 7.2 or <
 - mysql5.6 or <
 - php-cli
 - php-curl

## Dependencies
This project requires PHP8 or < .
* Whoops
   * Version: **2.14.4**
