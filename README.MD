## About Becwork
Becwork is a simple PHP framework.

## Documentation
Complete becwork documentation: [here](https://github.com/lordbecvold/becwork/blob/main/DOC.MD)

## Features
- Included composer
- Config system
- Mysql utils
- Crypt utils
- Hash utils
- Cookie utils
- Session utils
- APP panel
- Tests system

## Dependencies
This project requires PHP8 or < .
* Whoops
   * Version: **2.14.4**
   * [Github](https://github.com/filp/whoops)
